# 네크로맨서 서바이버 개발 프로젝트

> - **[최적화 및 구조 룰]**
>   1. JSON/PlayerPrefs 저장을 위한 Key 값들은 하드코딩하지 않고 전용 관리 스크립트(`Constants.cs` 또는 `KeyManager.cs` 등)에 상수로 일괄 선언하여 사용합니다.
>   2. 잦은 문자열 연산은 가비지 컬렉터 부하를 막기 위해 가급적 `StringBuilder`를 사용하고, TextMeshPro 등의 UI 텍스트 업데이트 시에는 메모리 할당 최소화를 위해 `.SetText()` 메서드를 활용합니다.
> - **코어 로직 1~2, 기믹 시스템, 전투 루프, 성장 시스템(레벨업 UI 및 3지선다 선택)**까지 작업 및 테스트가 완료되었습니다.
> - **[성장 스킬 데이터 연동 및 실제 적용 파이프라인]**을 완료했습니다. 20종 스킬 뽑기 및 `SkillManager` 를 통한 전역 미니언 스탯 버프(독, 결빙, 자석 사거리 등) 코드 연결을 마쳤습니다.
