## 🛠️ Core Principles
1. **Stability First**: Null 체크, defensive coding (Debug.LogError) 필수.
2. **Decoupled System**: Interface 및 Action 기반 이벤트 시스템 활용.
3. **Sleep Policy**: 씬별 매니저 생명주기 엄격 분리.
4. **Resource Integrity**: 재화는 오직 `ResourceManager`를 통해서만 접근.
